Simulation

Das ist eine Simulation einer Wiese, die von Kreaturen die Grass verschlingen zerlegt wird. Dabei gibt es
auch andere Kreaturen, die die Grassfressenden Kreaturen essen. Außerdem schlagen dauernd Meteroit-
en in die Landschaft ein, währens Flussbauer dauernd die Landschaft durchqueren und Flüsse bauen wollen.
Aktuell ist Sommer.
